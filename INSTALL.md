# Install Typing Master

Typing Master is a **PWA** (Progressive Web App). That means you install it
straight from the website — no app store, no download of a sketchy file — and
it then behaves like a normal app: its own icon, its own window, works offline,
and **all your progress, settings, and stats are saved on your own device.**

**Live app:** https://dilip-kumar-22.github.io/typing-master-scorp/

---

## 💻 Install on a laptop / desktop

### Chrome, Edge, or Brave
1. Open the live app link above.
2. Look in the address bar (right side) for an **install icon** — a little
   monitor with a down-arrow ⊕, or a ⊞ icon. Click it.
   *(Or use the in-app **Install** button in the top bar.)*
3. Click **Install**. Typing Master opens in its own window and gets a desktop /
   Start-menu shortcut.

### Safari (macOS)
1. Open the link.
2. **File → Add to Dock** (Safari 17+). It appears in your Dock as an app.

---

## 📱 Install on a phone / tablet

### Android (Chrome)
1. Open the link.
2. Tap the **⋮** menu → **Install app** (or **Add to Home screen**).
3. Confirm. The icon lands on your home screen.

### iPhone / iPad (Safari)
1. Open the link in **Safari** (not Chrome — iOS only allows Safari to install
   web apps).
2. Tap the **Share** button (the square with an up-arrow).
3. Scroll down → **Add to Home Screen** → **Add**.

---

## ✅ What you get

- **Offline use** — once installed, it works with no internet.
- **Your data stays yours** — progress, unlocked chapters, stats, themes, and
  settings are stored locally on your device (via the browser's storage). No
  account needed. *(Optional: sign in to sync across devices — that's a separate
  feature and requires a backend, see [docs/SUPABASE_SETUP.md](docs/SUPABASE_SETUP.md).)*
- **Auto-updates** — when the site updates, the installed app updates itself the
  next time you open it online.

---

## ❓ "I don't see an install button"

- Make sure you're on **https://** (the GitHub Pages URL), not a local file.
- On desktop, some browsers only show the install icon after a few seconds on
  the page. Reload once.
- Firefox desktop doesn't support PWA install — use Chrome/Edge/Brave, or just
  use it in the browser tab (it still works and still saves your progress).
